# Smart Downloads Folder Organizer

A Python automation tool that organizes files in the Downloads folder into categorized folders.

## Features

- Organizes files by type
- Supports Images, Documents, Videos, Music, and Archives
- Prevents filename collisions
- Generates activity logs
- Dry Run mode for safe testing

## Technologies

- Python
- pathlib
- shutil
- datetime

## Usage

`bash
python organizer.py
`

Set:

`python
DRY_RUN = False
`
to actually move files.
